# handwrittenMail
iOS版手邮APP简介及使用说明(How to use iOS App "handEmail" )

- 全功能IMAP和SMTP邮件客户端,支持126,163,Sina,QQ,Yahoo等主流邮件服务
- 邮件接收,邮件发送,邮件删除
- 新建,回复,转发邮件
- 邮件模板功能支持
- 邮件附件预览和第三方APP分享功能
- 新建,回复,转发邮件支持手写邮件
- 全功能手写邮件支持:写字,画线,画圆,画框,荧光笔,橡皮,插入图片,多页面,保存到相册,信纸背景切换,写字时笔迹粗细+颜色+pencil灵敏度设置.
- 手写邮件功能仅支持Apple Pencil.
